# Dataset

This repository uses two datasets:

1. SGCC Smart Meter Dataset
2. PMU Frequency Dataset

## Files

- sgcc_ml_ready.csv
- merged_frequency_data.csv

## Description

The SGCC dataset contains electricity consumption measurements and theft labels.

The PMU dataset contains synchronized frequency measurements used to improve electricity theft detection through feature fusion.

## Note

The datasets are not included in this repository due to licensing and data-sharing restrictions.

To reproduce the experiments, place the following files in this directory:

- sgcc_ml_ready.csv
- merged_frequency_data.csv
